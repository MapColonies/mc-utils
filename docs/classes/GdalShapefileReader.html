Class GdalShapefileReader

Adapter class that wraps gdal-async to provide a shapefile reader compatible with the interface expected by ShapefileChunkReader.

This adapter allows seamless migration from the 'shapefile' npm package to 'gdal-async' while maintaining the same API contract.
